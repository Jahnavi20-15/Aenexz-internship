This repository contains basic Python programs for beginners.  
The programs demonstrate the use of:

1.Conditional Statements
2.Loops
3.Pattern Printing
4.User Input
5.Basic Logic Building

Programs Included

1. Largest of Three Numbers
Finds the largest number among three given numbers using "if-elif-else".

2. Leap Year Checker
Checks whether a given year is a leap year or not.

3. Vowel or Consonant
Checks whether the entered character is a vowel or consonant.

4. Divisible by 5 and 11
Checks if a number is divisible by both 5 and 11.

5. Sum of First N Natural Numbers
Calculates the sum of first "N" natural numbers using a "while"  loop.

6. Multiplication Table
Prints the multiplication table of a given number using a "while" loop.

7. Pyramid Pattern.
Prints a star pyramid pattern.

8. Number Pattern
Prints the  number pattern.
